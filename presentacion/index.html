The Spyder Girl

Análisis y visualización de datos de las publicaciones en redes sociales

Creado por Geek & Tech Girls

@geekandtechgirl @geekandtechgirls

geekandtechgirl

¿Qué queremos conseguir?

Saber si las publicaciones de nuestras redes sociales están llegando al público que queremos, y conocer además qué momentos del día son mejores para realizar una publicación, en función, de la actividad de las publicaciones anteriores.

¿Cómo lo haremos?

  • Extrayendo datos de Twitter y Facebook.
  • Analizando los datos para extraer información sobre las publicaciones.
  • Realizando una visualización de datos sobre estos.
  • Además, tendremos un blog para complementar el proyecto.

¿Qué usaremos?

  • Python
  • API de twitter
  • API de facebook
  • D3.js

¿Cómo puedes ayudar?

  • Ayudando con el diseño del blog.
  • Documentando el proyecto y el blog.
  • Ayudando con las visualizaciones de datos con D3.js

¿Qué puedes aprender?

  • Manejo de GitHub y trabajo colaborativo
  • Python
  • Iniciarte en la minería de datos
  • Uso de las APIs de la redes sociales
